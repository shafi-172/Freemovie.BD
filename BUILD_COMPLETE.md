## 🎨 Customization Quick Reference

### Change Primary Color
**File**: `styles/main.css` (line 11)
```css
--primary-color: #FF6B35;  /* Change this to your color */
```

### Add Your First Movie
**File**: `js/data.js` (in moviesDatabase array)
```javascript
{
    id: 17,
    title: "Your Movie Title",
    rating: 8.5,
    year: 2024,
    genre: ["Your", "Genres"],
    image: "your-image-url",
    description: "Your description...",
    director: "Director Name",
    cast: ["Actor 1", "Actor 2"],
    duration: "120 min",
    budget: "$100M",
    boxOffice: "$500M",
    language: "English",
    releaseDate: "March 20, 2024",
    category: "recent"
}
```
